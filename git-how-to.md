1) ssh-keygen -t ed25519 -C "имя.фамилия@phystech.edu"
2) Откроется окно, в котором можно ввести название в поле Title. В поле Key нужно вставить публичный ключ. Его можно получить при помощи команды на вашем компьютере cat ~/.ssh/id_ed25519.pub. Скопируйте ключ из терминала, и вставьте его, после чего нажмите Add SSH key.
3) git clone <url>
